# lipu "kili.kipi.si"

https://kili.kipi.si

## pali

o kama jo [deno](https://deno.land/).

pini la, o kama jo [lume](https://lumeland.github.io/):

```
$ deno run -A https://deno.land/x/lume/install.ts
```

o pali e lipu:

```
$ git clone https://github.com/metanomial/kilikipisi.git
$ cd kilikipisi
$ lume
```
